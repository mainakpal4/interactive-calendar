# 📅 Interactive Calendar Component

## 🚀 Overview
This project is an interactive wall-style calendar built using HTML, CSS, and JavaScript. It allows users to select date ranges, add notes, and navigate across months with a clean and responsive UI.

---

## ✨ Features

- 📆 Real Calendar (Month + Year + Week Alignment)
- 🎯 Date Range Selection (Start → End → Highlighted)
- 📝 Notes System (Saved using localStorage)
- 🔄 Month Navigation (Previous / Next)
- 📱 Fully Responsive Design
- 🎨 Dynamic Theme Colors
- 🎬 Smooth Animations (Calendar transition)
- 📅 Holiday Markers
- 💾 Export Notes as JSON

---

## 🧠 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage API


## 🛠️ How to Run

1. Download / Clone the repository
2. Open `index.html` in browser
